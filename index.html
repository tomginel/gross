⚡ נדרשת הגדרת Firebase

  1. לך ל-console.firebase.google.com וצור פרויקט
  2. לחץ "Add app" ← סמל Web </> וקבל את firebaseConfig
  3. הפעל Realtime Database — בחר test mode
  4. פתח את הקובץ index.html בעורך טקסט
  5. מצא את firebaseConfig בסקריפט והחלף את שבעת הערכים
  6. שמור ורענן — הרשימה תתחיל לעבוד בין כל הטלפונים
